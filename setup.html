Welcome to Blanko!

Thank you for installing this extension. To enable window transparency, you'll need to make a small change in Zen's configuration:

  1. Open a new tab and navigate to about:config
  2. Search for: browser.tabs.allow_transparent_browser
  3. Set its value to true by clicking the toggle button
Note: If you decide to remove this extension later, remember to set browser.tabs.allow_transparent_browser back to false manually through about:config if you no longer need this feature.

You can always access this setup page by clicking the extension icon in your toolbar.
